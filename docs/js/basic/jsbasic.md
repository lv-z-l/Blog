### 1.1 JavaScript 简介

- JavaScript 是一种轻量级的编程语言。

- JavaScript 是可插入 HTML 页面的编程代码。

- JavaScript 插入 HTML 页面后，可由所有的现代浏览器执行。

- JavaScript 很容易学习。
- 由浏览器解释执行，会涉及到浏览器的兼容性问题。

### 1.2 JavaScript 能做什么

- 直接写入 HTML 输出流
- 对事件的反应
- 改变html的内容
- 改变html的样式
- 表单输入验证
